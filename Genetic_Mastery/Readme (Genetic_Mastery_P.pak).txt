Genetic_Mastery
----------------------------------------------------------------------
Mod Version: 1.0
Author: AgentKush
Compatible with Icarus: All Weeks
Mod Type: EXMOD (data)

## Description:
Elite trait lineages are 4x more common - wild and bred creatures far more often carry powerful genetics (Brave, Hardy, Bold, Fierce, Savage and more). Plain Wild and rare Alpha untouched, so it stacks with Genetics & Breeding Overhaul. Edits Weighting in AI-D_GeneticLineages.json.

Ver 1.0
Initial release. 10 trait lineages 4x spawn weighting.

## Installation:
  1. Install JimK72's Icarus Mod Manager
  2. Download Genetic_Mastery.EXMODZ
  3. Import via Mod Manager
