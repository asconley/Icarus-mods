## Changes

Every named trait lineage gets a **4× spawn weighting**:

| Lineage | Trait |
|---|---|
| Brave / Fierce / Savage | Melee damage |
| Hardy / Stout | Health & stamina |
| Bold / Resolute | Stamina / threat |
| Careful / Timid / Ambitious | Speed / utility |

The plain **Wild** lineage and the rare **Alpha** are left untouched, so this stacks cleanly with **Genetics & Breeding Overhaul** (which handles mutation talents and Wild/Alpha rarity).

## How It Works

Data mod that raises the `Weighting` on the named trait rows in `AI-D_GeneticLineages.json`. Different rows than Genetics & Breeding Overhaul — run both for the full genetics package.

## Installation

Install via JimK72's Icarus Mod Manager.
