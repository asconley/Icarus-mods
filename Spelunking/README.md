## Overview

Spelunking transforms cave exploration from a chore into a rewarding adventure! The Underground buff is enhanced to give better stamina bonuses, and the dreaded Cave Sickness (Pneumonia) is completely reworked into a powerful buff that makes you WANT to stay underground.

## Features

### Underground Buff (Enhanced)

When you enter a cave, you now receive "Spelunking!" instead of the basic Underground effect:

| Stat | Bonus |
|------|-------|
| Maximum Stamina | +50 |
| Stamina Action Cost | -10% |
| Stamina Regen | +25% |

*"You are in a cave, not Tartarus' butthole. Enjoy the extra stamina you magnificent bastard."*

### Cave Sickness → "Cave? Sick!" (Reworked)

Pneumonia is completely transformed from a debilitating sickness into an awesome buff:

| Stat | Bonus |
|------|-------|
| Maximum Stamina | +75% |
| Stamina Regen | +25% |
| XP Gain | +10% |

*"Caves are cool as fuck, you could stay down here forever!"*

### Key Benefits

- **No Pneumonia Penalty** - Cave Sickness is now a reward, not a punishment
- **Enhanced Exploration** - Better stamina means more mining and fighting
- **XP Bonus** - Gain 10% more experience while underground
- **Thematic Fun** - Humorous descriptions add character to the experience

## Installation

1. Download the `.EXMODZ` file from this repository
2. Open JimK72's Icarus Mod Manager
3. Import the mod file
4. Enable and launch Icarus
5. Head into any cave and enjoy the buffs!
