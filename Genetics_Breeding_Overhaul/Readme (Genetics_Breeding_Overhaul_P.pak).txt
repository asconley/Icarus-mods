Genetics_Breeding_Overhaul_P.pak
----------------------------------------------------------------------
Mod Version: 1.0

Author: AgentKush

Compatible with Icarus: All Weeks (requires the Genetics / Homestead system)

Mod Type: EXMOD

## Description:
Supercharges the Mendel genetics and selective-breeding system. The Genetics
talent tree rewards are doubled - genotype and phenotype mutation chance, twin
birth chance, gestation speed, fertilization recovery, lineage mutation, and the
"better wild creatures" chances. Bloodline rarity is rebalanced so the plain Wild
bloodline appears far less often (300 -> 75) and the prestige Alpha bloodline is
more achievable (10 -> 25). No base combat-stat changes, so it stacks with the
taming and difficulty mods.

Ver 1.0
Initial release. 2x genetics talent rewards across mutation, twins, gestation,
recovery, lineage and wild-genetics; Wild bloodline rarer, Alpha more achievable.

## Files Modified:
  Talents-D_Talents          (11 genetics talents)
  AI-D_GeneticLineages        (2 entries - Wild, Alpha)

## Installation:
  1. Install JimK72's Icarus Mod Manager
     https://github.com/jimk72/IcarusModManager
  2. Download Genetics_Breeding_Overhaul.EXMODZ
  3. Import via Mod Manager
