# Waste Not

![Version](https://img.shields.io/badge/version-2.0-blue)
![Author](https://img.shields.io/badge/author-CritFail%20(Updated%20by%20AgentKush)-green)
![Compatibility](https://img.shields.io/badge/compatibility-All%20DLCs-brightgreen)

## Overview

Waste Not makes mining dense ore veins more rewarding by replacing useless Stone drops with useful secondary resources. Dense metal ores now have a chance to drop Exotics, while other dense deposits give you more of their respective resource type.

## Features

### Dense Ore Changes

| Ore Type | Primary Resource | Secondary Resource | Effect |
|----------|------------------|-------------------|--------|
| Stone_Dense | Silica | - | Get Silica instead of Stone |
| Gold_Dense | Gold Ore | Meta_Resource | Chance of Exotics |
| Bauxite_Dense | Bauxite | Meta_Resource | Chance of Exotics |
| Metal_Dense | Iron Ore | Meta_Resource | Chance of Exotics |
| Copper_Dense | Copper Ore | Meta_Resource | Chance of Exotics |
| Oxite_Dense | Oxite | Oxite | Get more Oxite |
| Sulfur_Dense | Sulfur | Sulfur | Get more Sulfur |
| Silica_Dense | Silica | Silica | Get more Silica |
| Lithium_Ore_Dense | Lithium | Meta_Resource | Chance of Exotics |
| Ruby_Ore_Dense | Ruby | Meta_Resource | Chance of Exotics |

### Key Benefits

- **No More Wasted Inventory** - Dense veins no longer fill your inventory with Stone
- **Exotic Farming** - Metal-based dense ores can drop Exotics as secondary loot
- **Resource Efficiency** - Element-based ores give more of that element
- **Updated for Latest Content** - Includes Lithium and Ruby dense ores added in recent updates

## Installation

1. Download the `.EXMODZ` file from this repository
2. Open JimK72's Icarus Mod Manager
3. Import the mod file
4. Enable and launch Icarus
5. Find dense ore veins and enjoy the improved drops!
