# Waste Not

Enhanced mining rewards mod for Icarus.

**Author:** CritFail (Updated by AgentKush)  
**Version:** 2.0  

---

## Features

### Boosted "Waste Not" Talent
The "Waste Not" talent (renamed to "What's This?") now provides significantly increased secondary resource chances:

| Rank | Vanilla | Modded |
|------|---------|--------|
| 1    | 5%      | **10%** |
| 2    | 8%      | **20%** |
| 3    | 10%     | **30%** |

### Dense Ores Drop Exotics
Mining dense ore deposits now has a chance to yield Exotic materials as secondary resources:

- Gold (Dense) → Exotics
- Iron (Dense) → Exotics
- Copper (Dense) → Exotics
- Bauxite (Dense) → Exotics
- Titanium (Normal) → Exotics
- Platinum (Normal) → Exotics
- Lithium (Dense) → Exotics
- Ruby (Dense) → Exotics

### Normal Ores Drop Bonus Resources
Normal ore deposits drop extra resources of the same type:

- Gold (Normal) → Bonus Gold Ore
- Iron (Normal) → Bonus Iron Ore
- Copper (Normal) → Bonus Copper Ore
- Bauxite (Normal) → Bonus Bauxite
- Oxite (Normal/Dense) → Bonus Oxite
- Sulfur (Normal/Dense) → Bonus Sulfur
- Silica (Normal/Dense) → Bonus Silica
- Lithium (Normal) → Bonus Lithium Ore
- Ruby (Normal) → Bonus Ruby Ore

### Additional Changes
- Dense Stone deposits yield Silica instead of Stone
- Various talent tree UI position adjustments
- Anvil Bench T4 requires Fabricator talent

---

## Installation

1. Download `Waste_Not.EXMODZ`
2. Import into Icarus Mod Manager
3. Enable the mod and merge

## Compatibility

This mod modifies:
- `Talents-D_Talents.json`
- `World-D_VoxelSetupData.json`

May conflict with other mods that modify the same talent entries or voxel data.
