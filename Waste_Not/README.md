# Waste Not

Better secondary resources from mining in Icarus.

**Author:** CritFail (Updated by AgentKush)  
**Version:** 2.1  

---

## Features

Every mineable deposit now drops something useful as a secondary resource instead of Stone (or nothing). The "Waste Not" talent affects secondary resource drop chance.

### Dense Metal Ores → Exotics
| Deposit | Secondary |
|---------|-----------|
| Gold (Dense) | Exotics |
| Iron (Dense) | Exotics |
| Copper (Dense) | Exotics |
| Bauxite (Dense) | Exotics |
| Lithium (Dense) | Exotics |
| Ruby (Dense) | Exotics |

### Normal Ores → Bonus of Same Type
| Deposit | Secondary |
|---------|-----------|
| Gold (Normal) | Gold Ore |
| Iron (Normal) | Iron Ore |
| Copper (Normal) | Copper Ore |
| Bauxite (Normal) | Bauxite |
| Oxite (Normal) | Oxite |
| Sulfur (Normal) | Sulfur |
| Silica (Normal) | Silica |

### Rare Ores → Exotics
| Deposit | Secondary |
|---------|-----------|
| Titanium (Normal) | Exotics |
| Platinum (Normal) | Exotics |
| Lithium (Normal) | Exotics |
| Ruby (Normal) | Exotics |

### Coal & Ice (Previously No Secondary)
| Deposit | Secondary |
|---------|-----------|
| Coal (Normal) | Charcoal |
| Coal (Dense) | Charcoal |
| Ice (Normal) | Oxite |
| Ice (Dense) | Oxite |

### Other Changes
| Deposit | Secondary |
|---------|-----------|
| Stone (Normal) | Silica |
| Stone (Dense) | Silica |
| Oxite (Dense) | Oxite |
| Sulfur (Dense) | Sulfur |
| Silica (Dense) | Silica |

**Total: 26 deposit types modified**

## Installation

1. Download `Waste_Not.EXMODZ`
2. Import into Icarus Mod Manager
3. Enable the mod and merge
