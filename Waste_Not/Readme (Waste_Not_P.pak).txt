Mod Version: 2.2

Author: CritFail (Updated by AgentKush)

Compatible with Icarus: All Weeks

Mod Type: EXMOD

## Description:
Mining yields useful secondary resources instead of Stone. Normal ores give bonus of the same type, dense metal ores drop Exotics. Coal gives Charcoal, Ice gives Oxite. Covers all 26 mineable deposit types.

## Installation:
  1. Install JimK72's Icarus Mod Manager
     https://github.com/jimk72/IcarusModManager
  2. Download Waste_Not.EXMODZ
  3. Import via Mod Manager
