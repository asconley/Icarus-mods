## Changes

| Property | Vanilla | Modded |
|---|---|---|
| Resource weight | 100% | **50%** |

Covers **54 resources** identified by the `Item.Resource` tag — every ore, ingot, refined metal and raw material (Stone, Wood, Fiber, Bone, Leather, Sulfur, Oxite, Silica, Coal, Clay, Sand, Wool, Fur and more). Tools, weapons, armour, building pieces and food are untouched.

## How It Works

Data mod that halves the `Weight` value in `Traits-D_Itemable.json` for every resource-tagged item. Pairs perfectly with Stack Size Overhaul.

## Installation

Install via JimK72's Icarus Mod Manager.
