Lightweight_Hauling
----------------------------------------------------------------------
Mod Version: 1.0
Author: AgentKush
Compatible with Icarus: All Weeks
Mod Type: EXMOD (data)

## Description:
All ores, ingots and raw resources weigh 50% less - haul far more per trip. Covers 54 resource-tagged items (ores, ingots, refined metals, raw materials). Tools, armour, building pieces and food are untouched. Edits Weight in Traits-D_Itemable.json. Pairs well with Stack Size Overhaul.

Ver 1.0
Initial release. 50% weight reduction across 54 resources.

## Installation:
  1. Install JimK72's Icarus Mod Manager
  2. Download Lightweight_Hauling.EXMODZ
  3. Import via Mod Manager
