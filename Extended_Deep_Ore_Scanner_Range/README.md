## Changes

| Property | Vanilla Range | Modded Range |
|---|---|---|
| Deep Ore Scan Range | 300m | **3000m (3km)** |

Applies to both the **Deep Ore Scanner** and the **Advanced Deep Ore Scanner** (the Advanced variant inherits the base scan range).

## How It Works

PAK mod that patches `BP_ActionableBehaviour_Scanner_DeepOre` to extend the `MaxScanningRange` value (30000 -> 300000 cm). Same-size in-place blueprint byte-patch; no other values are touched.

## Installation

Install via JimK72's Icarus Mod Manager.

## Multiplayer

All players AND the server must have this mod installed.
