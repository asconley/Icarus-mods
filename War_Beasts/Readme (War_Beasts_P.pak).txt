War_Beasts
----------------------------------------------------------------------
Mod Version: 1.0
Author: AgentKush
Compatible with Icarus: All Weeks
Mod Type: EXMOD (data)

## Description:
The first taming-combat mod for Icarus. Husbandry combat talents 3x stronger (Kill Command damage 3x, Hardy Diet health 3x, Like the wind speed 2x). Spec the Husbandry tree to build true war beasts. Edits talent reward values in Talents-D_Talents.json - no base-stat edits, never conflicts.

Ver 1.0
Initial release. Husbandry combat talents boosted.

## Installation:
  1. Install JimK72's Icarus Mod Manager
  2. Download War_Beasts.EXMODZ
  3. Import via Mod Manager
