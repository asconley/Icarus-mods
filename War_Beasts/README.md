## Changes

| Talent | Effect | Boost |
|---|---|---|
| Kill Command | Tamed melee damage | **3×** |
| Hardy Diet | Tamed health + regen | **3×** |
| Like the wind | Mount speed | **2×** |

A fully-specced Husbandry build now produces genuinely combat-viable mounts and companions instead of fragile pack animals. Spec the talents to unlock the power — balanced, not a free buff.

## How It Works

Data mod that triples the `GrantedStats` reward values on the `Husbandry_Damage`, `Husbandry_TameHealth` and `Husbandry_MountSpeed` talents in `Talents-D_Talents.json`. No base-stat edits, so it never conflicts with other mods.

## Installation

Install via JimK72's Icarus Mod Manager.
