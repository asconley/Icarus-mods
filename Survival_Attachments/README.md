Adds food, oxygen, and water armor attachments. Doubles attachment slots on every armor piece so you can stack survival mods with existing combat or yield mods.

Requires **[JimK72's Icarus Mod Manager](https://github.com/Jimk72/Icarus_Software)**

</div>

---

## Features

- **Doubles attachment slots** on every armor piece (Head / Arms / Body / Legs / Feet) from 1 to 2
- **Three new craftable attachments** at the Alteration Bench that plug into the standard armor attachment system
- Uses existing base-game stat keys (`BaseOxygenConsumption_+%`, `BaseFoodConsumption_+%`, `BaseWaterConsumption_+%`) so effects stack correctly with talents and other gear
- No PAK file required - pure data-table mod

## New Attachments

| Attachment | Slot | Effect |
|------------|------|--------|
| Oxygen Recycler | Head | -25% oxygen consumption |
| Nutrient Dispenser | Body | -25% food consumption |
| Hydration Cell | Legs | -25% water consumption |

Arms and Feet pieces also gain a second slot but no new attachment type - use them for existing attachments (Melee Damage, Tool Use, Fall Resistance, Extra Pockets, etc.).

## Crafting Recipes

## Installation

1. Download `Survival_Attachments.EXMODZ`
2. Import into **Icarus Mod Manager** (JimK72's IMM)
3. Enable and merge mods as usual
