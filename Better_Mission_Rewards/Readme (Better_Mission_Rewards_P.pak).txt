Better_Mission_Rewards_P.pak
----------------------------------------------------------------------
Mod Version: 1.0

Author: AgentKush

Compatible with Icarus: All Weeks

Mod Type: EXMOD

## Description:
Doubles mission payouts so you grind fewer missions for the same progress.
Credits, account experience, and faction reputation are all 2x across every
faction mission. Talent unlocks (TalentsRewarded) are left untouched so nothing
is skipped or broken.

Ver 1.0
Initial release. 2x credits, 2x account XP, 2x faction reputation across all
faction missions.

## Files Modified:
  Factions-D_FactionMissions   (all reward-bearing missions)

## Installation:
  1. Install JimK72's Icarus Mod Manager
     https://github.com/jimk72/IcarusModManager
  2. Download Better_Mission_Rewards.EXMODZ
  3. Import via Mod Manager
