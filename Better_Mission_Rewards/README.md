## Features

- **2× Credits** — `CurrencyRewarded` doubled on every mission that pays Ren/Credits.
- **2× Account XP** — faster account-level progression.
- **2× Faction Reputation** — rank up with the factions in half the missions.
- **Talent unlocks untouched** — `TalentsRewarded` is left as-is, so nothing is skipped or broken.

Covers all faction missions across every map (Olympus, Styx, Prometheus, and DLC).

## Files Modified

- `Factions-D_FactionMissions` (all reward-bearing missions)
