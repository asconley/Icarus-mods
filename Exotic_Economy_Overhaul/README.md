# Exotic Economy Overhaul

A full currency exchange system using Exotics. Convert between exotic types, buy rare materials, and sell surplus resources — all at the Fabricator.

## Features

### Exotic Cross-Conversion (12 recipes)
Convert any exotic type to any other at a 2:1 ratio:
- Desert ↔ Arctic ↔ Swamp ↔ Forest Exotics

### Buy Materials with Exotics (15 recipes)
Spend generic Exotics to purchase rare crafting materials:
- Electronics (5 for 20 Exotics)
- Carbon Fiber, Epoxy, Steel/Titanium/Platinum/Gold/Copper/Aluminum Ingots
- Composites, Glass, Concrete Mix, Gunpowder, Steel Screws, Refined Metal

### Sell Resources for Exotics (7 recipes)
Convert surplus ores and materials into Exotics:
- Gold Ore (50 → 5 Exotics), Platinum Ore (50 → 8), Titanium Ore (50 → 10)
- Steel Ingots (20 → 5), Electronics (10 → 8), Composites (10 → 5)
- Diamond Ore (5 → 15 Exotics)

**34 total recipes** available at Fabricator and T4 Fabricator.

## Installation

Import `Exotic_Economy_Overhaul.EXMODZ` via Icarus Mod Manager.
