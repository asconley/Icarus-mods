Requires **[JimK72's Icarus Mod Manager](https://github.com/Jimk72/Icarus_Software)**

## Features

### Pickaxe Upgrades — 4 recipes
Stone -> Metal -> Steel -> Platinum -> Titanium

### Axe Upgrades — 4 recipes
Stone -> Metal -> Steel -> Platinum -> Titanium

### Knife Upgrades — 4 recipes
Stone -> Metal -> Steel -> Platinum -> Titanium

### Bow Upgrades — 3 recipes
Wood Bow -> Recurve Bow -> Longbow -> Compound Bow

### Crossbow Upgrades — 2 recipes
Crossbow -> Platinum Crossbow -> Titanium Crossbow

### Specialty Paths — 9 recipes
Branch off the main upgrade tree into specialty materials:
- **Obsidian** (from Steel tier): Pickaxe, Axe, Knife
- **Cold Steel** (from Platinum tier): Pickaxe, Axe, Knife
- **Iron Wood** (from Titanium tier): Pickaxe, Axe, Knife

## How It Works

Each recipe takes the previous tier weapon + upgrade materials. Your old tool becomes the new one — no wasted resources crafting from scratch.

## Compatibility

- ✅ Compatible with all other mods

## Installation

Import `Tier_Upgrade_Forge.EXMODZ` via Icarus Mod Manager.
