# Tier Upgrade Forge

Upgrade your tools and weapons to the next tier without rebuilding from scratch. Feed your current tool plus upgrade materials into the appropriate workbench and get the next tier item.

## Features

- **26 upgrade recipes** across 6 weapon/tool categories
- **Pickaxes**: Stone → Bone → Metal → Steel → Platinum → Titanium
- **Axes**: Stone → Bone → Metal → Steel → Platinum → Titanium
- **Knives**: Stone → Bone → Metal → Steel → Platinum → Titanium
- **Spears**: Wood → Bone → Metal → Steel → Platinum → Titanium
- **Swords**: Metal → Steel → Platinum → Titanium
- **Bows**: Wood → Long → Recurve → Compound

## How It Works

Each upgrade recipe takes:
1. The current tier item (1x)
2. The upgrade material (varies by tier)
3. Leather binding (2x for bench-crafted upgrades)

Recipes appear at the same workbench where the target tier item is normally crafted.

## Installation

Import `Tier_Upgrade_Forge.EXMODZ` via Icarus Mod Manager.
