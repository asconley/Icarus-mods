# Security Policy

## Supported Versions

Only the latest version of each mod is supported. Older versions do not receive security updates.

| Version | Supported |
|---------|-----------|
| Latest release | Yes |
| Previous releases | No |

## Reporting a Vulnerability

If you discover a security issue in any mod file (malicious code, compromised assets, etc.), please report it responsibly:

1. **Do not** open a public issue
2. **Email** finlayhansford66@gmail.com with details of the vulnerability
3. Include which mod and version is affected
4. Describe the potential impact

You should receive a response within 48 hours. Critical issues will be patched and released as quickly as possible.

## Mod Safety

All mods in this repository are JSON-based data modifications (EXMOD format) or Unreal Engine PAK overrides. They do not contain executable code, scripts, or network calls. Every mod is open-source and can be inspected by extracting the EXMODZ (zip) file.

The automated validation workflow checks all submitted mods for structural integrity and correct formatting before they are merged.
