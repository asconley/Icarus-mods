# Hardcore Rebalance Pack

For survivors who think Icarus is too easy. This mod cranks up the difficulty across every survival system with 348 changes across 6 data tables.

## Changes

### AI & Spawns
- **AI Spawn Zones** (160 zones): All creature spawn weights +50% — more animals everywhere
- **Autonomous Spawns** (22 entries): +60% max spawn count, +50% spawn distance from players
- **Horde Waves** (21 waves): +60% total spawns, +50% simultaneous spawns, -40% delay between waves, +50% extra spawns per nearby player

### Environment
- **Weather Events** (109 events): All storm durations reduced 40% — storms cycle faster, less downtime between them
- **Decay Timers** (32 items): All decay halved — food spoils faster, bodies disappear quicker
- **Fuel Efficiency** (4 items): All fuel millijoules halved — burn through resources twice as fast

## ⚠️ Warning

This mod is designed to make the game significantly harder. Not recommended for new players. Consider pairing with extra carry weight or resource mods to offset the increased resource drain.

**Not compatible with Performance Optimizer** — they modify the same tables in opposite directions.

## Installation

Import `Hardcore_Rebalance_Pack.EXMODZ` via Icarus Mod Manager.
