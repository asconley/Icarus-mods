## Changes

| Tool action | Vanilla | Modded |
|---|---|---|
| Mining efficiency (pickaxes) | 1× | **2×** |
| Felling efficiency + damage (axes) | 1× | **2×** |
| Reaping efficiency (sickles) | 1× | **2×** |
| Skinning efficiency (knives) | 1× | **2×** |

Applies to **every tier** of pickaxe, axe, sickle and knife (Stone → Cold Steel, plus specialty tools).

## How It Works

Data mod that doubles the `*_Efficiency` values (and axe `Felling_Damage`) in `Tools-D_ToolDamage.json`. Stacks with Creature Drop Enhancer for skinning.

## Installation

Install via JimK72's Icarus Mod Manager.
