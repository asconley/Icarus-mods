Faster_Harvest_Tools
----------------------------------------------------------------------
Mod Version: 1.0
Author: AgentKush
Compatible with Icarus: All Weeks
Mod Type: EXMOD (data)

## Description:
2x harvest yield from every gathering tool - mining, felling, reaping and skinning - plus faster tree felling. Doubles the *_Efficiency values (and axe Felling_Damage) in Tools-D_ToolDamage.json for every tier of pickaxe, axe, sickle and knife. Stacks with Creature Drop Enhancer.

Ver 1.0
Initial release. 2x mining/felling/reaping/skinning efficiency.

## Installation:
  1. Install JimK72's Icarus Mod Manager
  2. Download Faster_Harvest_Tools.EXMODZ
  3. Import via Mod Manager
