The **first-ever critical-hit mod** for Icarus. Weak-point hits deal far more critical damage — reward precise aiming when hunting and fighting.

Requires **[JimK72's Icarus Mod Manager](https://github.com/Jimk72/Icarus_Software)**

## Changes

| Weak point | Vanilla crit | Modded crit |
|---|---|---|
| Standard weak point | 0.5× (+150%) | **1.0× (+300%)** |
| Elite weak point | 1.0× (+300%) | **1.5× (+450%)** |
| Mini weak point | 0.15× (+45%) | **0.4× (+120%)** |
| Metal weak point | 0.5× | **1.0×** |

Crit damage scales the base `CriticalDamage_+%` stat (300%) by these per-zone multipliers. Strong/hardened points (armour) are untouched — this only rewards hitting weak spots.

## How It Works

Data mod that raises the `MultiplierStatMultiplier` of the weak-point rows in `Damage-D_CriticalHitAreas.json`. Nobody has modded the critical-hit system before.

## Installation

Install via JimK72's Icarus Mod Manager.
