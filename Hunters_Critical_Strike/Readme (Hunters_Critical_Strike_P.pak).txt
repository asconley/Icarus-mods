Hunters_Critical_Strike
----------------------------------------------------------------------
Mod Version: 1.0
Author: AgentKush
Compatible with Icarus: All Weeks
Mod Type: EXMOD (data)

## Description:
The first-ever critical-hit mod for Icarus. Weak-point hits deal far more critical damage - standard and metal weak points 2x, mini ~2.6x, elite +50%. Strong/hardened armour points untouched. Edits MultiplierStatMultiplier in Damage-D_CriticalHitAreas.json.

Ver 1.0
Initial release. Boosted weak-point crit multipliers.

## Installation:
  1. Install JimK72's Icarus Mod Manager
  2. Download Hunters_Critical_Strike.EXMODZ
  3. Import via Mod Manager
