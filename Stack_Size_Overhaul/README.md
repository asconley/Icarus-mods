## Changes

| Category | Multiplier | Examples |
|----------|-----------|---------|
| Resources/Ammo/Ores | **5x** | Fiber 200→1000, Stone 100→500, Arrows 100→500, Ore 50→250, Seeds 400→2000 |
| Building Pieces | **3x** | Floors/Walls 20→60, Furniture 5→15 |
| Consumables | **3x** | Bandages 10→30, Grenades 5→15 |
| Food/Drinks | **3x** | Berries 100→300, Meals 10→30 |
| All others | **3x** | Max cap 9999 |

328 items at 5x, 934 items at 3x. Unstackable items left at 1.

## Installation

Import `Stack_Size_Overhaul.EXMODZ` via Icarus Mod Manager.
