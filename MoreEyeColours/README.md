# More Eye Colours

Massive eye color expansion for Icarus character creation, plus unlocked hidden content.

## Features

### Unlocked Content (3)
- Spacesuit Male & Female (normally hidden)
- Deluxe Edition suit color (no DLC required, UI swatch fixed)

### Eye Colors (+60 new, 69 total)
Greens: Emerald, Forest, Jade, Lime, Olive, Mint, Pine, Seafoam, Chartreuse
Warm/Amber: Amber, Honey, Copper, Gold, Topaz, Caramel, Bronze, Hazel Gold
Purples: Violet, Amethyst, Lavender, Rose, Plum, Orchid, Royal Purple, Mauve, Lilac
Blues: Ice Blue, Electric, Sapphire, Teal, Cyan, Cobalt, Sky, Ocean, Midnight, Aquamarine, Periwinkle, Turquoise
Greys: Steel, Silver, Charcoal, Ash, Smoke, Slate, Gunmetal
Reds: Crimson, Ruby, Ember, Molten, Blood Red, Garnet, Rust, Scarlet
Exotic: White, Neon Green, Neon Pink, Neon Cyan, Neon Orange, Pitch Black, Pale Yellow

## Installation
Drop the `.EXMODZ` file into your Icarus Mod Manager mods folder.

## Compatibility
Modifies `D_CharacterCreationData.json` only. Compatible with most other mods.
