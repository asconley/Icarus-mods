# 🏗️ Hidden Building Pieces

Unlocks **810 hidden building items** including diagonal/curved pieces, frames, corner stairs, quarter floors, and roof peak pieces for Icarus.

## Features

### Diagonal & Curved Pieces (9 Materials)
- **Materials:** Concrete, Glass, Clay Brick, Stone, Scoria, Tempered Glass, Stone Brick, Scoria Brick, Ice
- **6 Piece Types:** Diagonal Set, Curved Set, Wall Curved, Curved Wall Angles, Roof Curved Angles, Advanced Beam Set

### Frame Pieces (7 Materials)
- **Materials:** ClayBrick, Ice, Glass, TemperedGlass, StoneBrick, ScoriaBrick, Scoria
- **3 Piece Types:** FullFrame, HalfFrame, Pillar

### Corner Stairs (ALL 18 Materials)
Every material that has stairs now has corner stairs:
- Wood, Thatch, Clay Brick, Ice, Scoria, Glass, TemperedGlass
- ScoriaBrick, Limestone, StoneBrick, Sandworm, ReinforcedWood, MetalGrate
- *(Stone, Concrete, Iron, Interior_Wood, Aluminium already had corner stairs in base game)*

### Roof Peak Connectors & Caps (15 Materials) - NEW in v2.10.0
Unlocks the hidden roof peak connector and cap pieces for ALL materials:
- **Materials:** Wood, Stone, Concrete, Thatch, Iron, Glass, Clay Brick, Scoria, Scoria Brick, Ice, Limestone, Stone Brick, Sandworm, Tempered Glass, Reinforced Wood
- **2 Piece Types:** Connector (joins roof peaks together), Cap (ends roof peaks)

### Quarter Floor (Metal Grate)
- Metal Grate now has a quarter floor piece matching other materials

## Installation

### Via Mod Manager (Recommended)
1. Add this repository URL to the Mod Manager's modinfo sources
2. Find "Hidden Building Pieces" and install
3. Click Merge to apply

### Manual Installation
1. Download `Hidden_Building_Pieces.EXMODZ`
2. Place in: `Icarus_Mod_Manager_2_x_x/Extracted_Mods/`
3. Use the Mod Manager to merge and install
