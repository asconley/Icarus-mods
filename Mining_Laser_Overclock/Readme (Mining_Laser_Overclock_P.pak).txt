Mining_Laser_Overclock
----------------------------------------------------------------------
Mod Version: 1.0
Author: AgentKush
Compatible with Icarus: All Weeks
Mod Type: EXMODZ (PAK / blueprint)

## Description:
The first-ever mining-laser blueprint mod. The mining laser runs 300 seconds before overheating (was 5s) and cools in 1s (was 5s) - sustained, uninterrupted deep mining. Byte-patches SecondsToReachMaxHeat and SecondsToCooldown in BP_Actionable_MiningLaser.

Ver 1.0
Initial release.

## Installation:
  1. Install JimK72's Icarus Mod Manager
  2. Import Mining_Laser_Overclock.EXMODZ
  3. Rebuild the merged mod and restart Icarus
