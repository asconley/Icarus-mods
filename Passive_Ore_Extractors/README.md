# Passive Ore Extractors

**Author:** AgentKush  
**Version:** 1.0  
**Mod Manager:** JimK72's Mod Manager (EXMODZ format)

## Description

Place ore extractor machines anywhere to passively generate ore over time. Cloned from Deep Mining Drills with processing capability added. Includes Biofuel (T3) and Electric (T4) variants with 23 ore types each. Available from the Workshop and craftable at the Fabricator.

## Machines

| Machine | Tier | Power Source | Workshop Cost | Crafted At |
|---------|------|-------------|---------------|------------|
| Biofuel Ore Extractor | T3 | Biofuel | 100R / 50C | Machining_Bench, Fabricator, Manufacturer |
| Electric Ore Extractor | T4 | Electricity | 150R / 75C | Fabricator, Manufacturer |

## Crafting Costs

### Biofuel Ore Extractor

- 12x Steel Ingot
- 10x Refined Copper
- 12x Epoxy
- 10x Electronics
- 50x Copper Wire

### Electric Ore Extractor

- 25x Aluminium Ingot
- 15x Electronics
- 12x Steel Screw
- 8x Titanium Ingot
- 8x Carbon Fiber

## Installation

1. Install JimK72's Mod Manager
2. Place the `.EXMODZ` file in your Mod Manager mods folder
3. Enable the mod and launch Icarus

## Notes

- Machines use the Deep Mining Drill models but function as processors
- Biofuel variant requires biofuel to run (same as biofuel drill)
- Electric variant requires electricity connection
- Both machines can be placed anywhere (no ore deposit required)
- Recipes produce ore passively using only power (no material input)
