# Passive Ore Extractors

**Author:** AgentKush  
**Version:** 2.0  
**Mod Manager:** JimK72's Mod Manager (EXMODZ format)

## Description

23 passive ore extractors — one for each ore type. Place anywhere, connect power, and ore is generated automatically. No interaction needed. Available from the Workshop and craftable at the Fabricator.

## How It Works

Each machine is dedicated to one ore type. Place it, connect electricity, and it automatically produces ore. No interaction needed after placement.

## Installation

1. Install JimK72's Mod Manager
2. Place the `.EXMODZ` file in your Mod Manager mods folder
3. Enable the mod and launch Icarus

## Notes

- All machines are electric-powered (connect to power grid)
- AutoSelectRecipe is enabled — machines start producing immediately when powered
- Workshop tab: 'Passive Miners'
