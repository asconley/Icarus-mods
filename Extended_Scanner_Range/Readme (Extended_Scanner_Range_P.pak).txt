Mod Version: 1.0

Compatible with Icarus: All Weeks
Mod Type: EXMOD + Blueprint Overrides

## Description:
10x detection range for the Cave Scanner and Fish Finder. Cave Scanner 300m -> 3000m, base/Fish Finder scanner 150m -> 1500m. PAK mod - patches MaxScanningRange in BP_ActionableBehaviour_Scanner_Cave and BP_ActionableBehaviour_Scanner. Companion to Extended Deep Ore Scanner Range.

Ver 1.0
Initial release. 10x cave + fish finder scanner range.

## Installation:
  1. Install JimK72's Icarus Mod Manager
  2. Download Extended_Scanner_Range.EXMODZ
  3. Import via Mod Manager
