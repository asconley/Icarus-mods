## Description:
Enhanced armor set bonuses with 3-piece partial bonuses for mix-and-match builds. 33 sets get a new 3-piece threshold (~40% of full bonus) plus boosted 5-piece bonuses (+75% positive stats, +50% resistances). Wear 3 pieces of any set for a partial bonus, complete the set for full power.


Ver 2.0
Added 33 three-piece partial set bonuses for mix-and-match builds. Each set now has two bonus thresholds: 3-piece and 5-piece.

Ver 1.0
Initial release enhancing all 33 armor set bonuses.
