# Armor Set Bonuses Enhanced v2.0

Enhanced armor set bonuses with 3-piece partial bonuses for mix-and-match builds. Wear 3 pieces of any set for a partial bonus, complete the full 5 pieces for maximum power.

## Changes

- **3-piece partial bonus (NEW):** ~40% of full set bonus with 1-2 key stats
- **5-piece full bonus:** +75% positive stats, +50% resistances (from v1.0)

## Mix-and-Match Examples

- **3 Polar Bear + 2 Composite** — Arctic speed bonus + strong health/stamina regen pieces
- **3 CX-400 + 2 Naneo** — Mining rewards bonus + weight capacity armor pieces
- **3 Bone + 2 Scorpion** — Melee damage bonus + projectile resist armor pieces
- **3 Hazmat + 2 Caveworm** — Exposure protection + mining rewards in caves

## Installation

Import `Armor_Set_Bonuses_Enhanced.EXMODZ` via Icarus Mod Manager.
