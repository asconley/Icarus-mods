# Armor Set Bonuses Enhanced

Supercharges all 33 armor set bonuses in the game. Wearing a full matching set now provides significantly stronger benefits.

## Changes

- **Positive stats** (speed, regen, damage, etc.): Boosted **+75%**
- **Resistance/reduction stats** (threat reduction, consumption reduction): Improved **+50%**

### Examples

| Armor Set | Original Bonus | Enhanced Bonus |
|-----------|---------------|----------------|
| Cloth | +5% Movement Speed | +9% Movement Speed |
| Leather | +10% Health/Stamina Regen | +18% Health/Stamina Regen |
| Polar Bear | +10% Arctic Move Speed, -20% Arctic Threat | +18% Arctic Move Speed, -30% Arctic Threat |
| Scorpion | +20% Projectile Resist, 40% Damage Return | +35% Projectile Resist, 70% Damage Return |
| CX-400 | +25% Mining/Felling Rewards | +44% Mining/Felling Rewards |

All 33 vanilla armor sets are enhanced.

## Installation

Import `Armor_Set_Bonuses_Enhanced.EXMODZ` via Icarus Mod Manager.
