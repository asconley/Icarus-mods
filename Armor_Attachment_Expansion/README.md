Adds **25 new armor attachments** (5 per body part) that slot into the standard armor attachment system. All effects use base-game stat keys so they stack cleanly with talents, food buffs, and existing attachments. Pairs perfectly with **Survival_Attachments** which doubles the number of available slots.

Requires **[JimK72's Icarus Mod Manager](https://github.com/Jimk72/Icarus_Software)**

</div>

---

## Features

- **25 new craftable attachments** spread evenly across Head / Arms / Body / Legs / Feet
- Every attachment uses a **verified base-game stat key**, so effects stack correctly
- **Uniform crafting cost** (2 Steel Ingot + 2 Epoxy + 1 Electronics) — easy to slot into any playthrough
- Crafted at the **Alteration Bench** or **Advanced Alteration Bench**
- No PAK file required — pure data-table mod

## Crafting Recipe

All 25 attachments share the same recipe:

| Material | Quantity |
|----------|----------|
| Steel Ingot | 2 |
| Epoxy | 2 |
| Electronics | 1 |

Crafted at the **Alteration Bench** or **Advanced Alteration Bench**.

## Installation

1. Download `Armor_Attachment_Expansion.EXMODZ`
2. Import into **Icarus Mod Manager** (JimK72's IMM)
3. Enable and merge mods as usual

## Pairs Well With

- **Survival_Attachments** - doubles armor attachment slots so you can run more of these at once
- Any Icarus talent build — these slot neatly under the existing Rank 1 attachment filter
